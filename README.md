# Full-Stack Listing Platform (MERN)
A full-stack MERN web application for listing and discovering properties with authentication, reviews, image uploads, and map-based location display.

## Features
- User authentication and authorization
- Create, edit, and delete property listings
- Image upload using Cloudinary
- Location mapping with Mapbox
- Reviews and ratings system
- Search listings by destination
- Responsive UI (desktop & mobile)
## Tech Stack
- Frontend: HTML, CSS, JavaScript, EJS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Passport.js
- Image Storage: Cloudinary
- Maps: Mapbox
## Installation
1. Clone the repository
2. Install dependencies using npm install
3. Add environment variables (MongoDB, Cloudinary, Mapbox)
4. Run the app using npm start
